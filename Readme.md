# 智能拷贝工具

## 1.主要功能：

1. 将源文件夹的内容拷贝到目标文件夹中，需要根据下述内容筛选：
   
   - 是否递归子文件夹
   - 文件夹的名称（可以为正则表达式）
   - 文件类型（后缀名，可以手工输入多个）
   - 文件大小（可以指定范围）
   - 空文件夹时是否拷贝

2. 源文件夹、目标文件夹支持拖曳选择。

## 2. 打包命令

- 生成单文件格式
  `pyinstaller -i liug.ico -F -w smart_copy.py --clean -n 智能拷贝工具`
